# Metadatos de los cuadros del Censo de Argentina

Un dataset que contiene la relación entre jurisdicciones, temas y
archivos para facilitar la descarga y visualización interactiva.

## Usage

``` r
info_cuadros_arcenso
```

## Format

Un data frame con 48 filas y 5 columnas:

- Jurisdiccion:

  Nombre de la provincia o jurisdicción (ASCII).

- tema:

  Categoría temática del censo.

- Titulo:

  Título descriptivo del cuadro estadístico.

- Archivo:

  Nombre del archivo asociado para descarga.

- Anio:

  Año del censo correspondiente.
